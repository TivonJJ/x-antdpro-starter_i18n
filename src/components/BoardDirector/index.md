# 带图标数字展示  
 
### 参数说明

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| icon | 图标 |  String | - |
| iconTheme | 图标主题 | outlined | - |
| title | 标题 | Any | - |
| color | 图标颜色 | String | - |
| extra | 补充内容 | Any | - |
| iconSize | 图标尺寸 | Number | 46 |
| contentClass | 内容区域的class | String | - |
